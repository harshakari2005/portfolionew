# Harsha Cyber Portfolio — JavaScript Version

This version is fully generated and controlled by `app.js`.

## Files

- `index.html` — minimal loader
- `app.js` — complete website structure, styling, data, terminal commands and animations
- `assets/harsha-profile.png`
- `assets/Harsha_Resume.pdf`

## Run

Double-click `index.html`, or use a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment

Upload the entire folder to GitHub Pages, Netlify, Vercel or any static host.

There is no backend and no build step.
